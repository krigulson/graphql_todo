# graphql_todo

## Build Setup

```bash
# clone repository
$ git clone https://github.com/krigulson/graphql_todo.git

# cd
$ cd graphql_todo

# install dependencies
$ yarn install

# create env file
$ touch .env

# open .env file and add API_ENDPOINT variable, for me its http://localhost:5000/graphql
$ code .env

# serve with hot reload at localhost:3000
$ yarn dev
```
